BootCell

BootCell

Web Components UI library based on WebCell v2, BootStrap v4, BootStrap Icon v1 & FontAwesome v5

Usage

 
npm install boot-cell \
    iterable-observer \
    marked turndown turndown-plugin-gfm markdown-ime \
    quill quill-image-uploader \
    @nuintun/qrcode
